*Installation*
 install dependencies: npm install or yarn install

*Local Setup*  
---------------
1. Setup Backend  
2. cd backend  
3. cp .env.example .env  # Add your own environment as below 
4. npm install

*Environment*

JWT_SECRET=your_secret
